This package contains the basic styles that draw points, shapes, etc on a graphics canvas. These styles contain some basic style information, and have the capability to render either a single shape or multiple shapes. The four basic styles are {@link org.blaise.style.ShapeStyle} for {@code java.awt.Shape}s, {@link org.blaise.style.PathStyle} for {@code java.awt.Shape}s representing paths, {@link org.blaise.style.PointStyle} for {@code java.awt.geom.Point2D}s with/without orientations, and {@link org.blaise.style.TextStyle} for drawing {@code String}s at a {@code java.awt.geom.Point2D}.

Style classes obey the following conventions:

  • classes are not final
  • classes have a unique toString() method
  • classes follow Java Bean conventions: have a no-argument constructor, and getter-setter methods for all their properties.
  • property fields are {@code protected}
  • there is builder pattern for each property, which uses the corresponding {@code set} method

Style objects have properties that match SVG styling rules (http://www.w3.org/TR/SVG/styling.html) as much as possible, with styles like {@code font-size} represented by CamelCase attributes/properties like {@code fontSize}.
